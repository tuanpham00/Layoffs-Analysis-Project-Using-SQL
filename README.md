# Layoffs Analysis Project Using SQL
# Overview
This project involves a comprehensive analysis of global mass layoff data from 2020-2023 using SQL. The goal is to clean and extract valuable insights and answer various business questions based on the dataset.

# Objectives
- Data cleaning: removing duplicate data and blank cells, standardizing data.
- Analyze the number of mass layoffs by country, industry, company, etc.
- Identify the ranking of companies with the highest number of layoffs each year.
